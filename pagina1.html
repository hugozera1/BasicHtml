HTML BASIC

Esse capitulo vai mostrar alguns conceitos basicos para se começar a aprender html.

No inicio de todo documento html deve se declarar o tipo do documento: <!DOCTYPE html> Logo após declarar começa com um: <html> e a parte visivel do codigo entra na tag <body>e que fecha </body>e termina com um: </html>


EXEMPLO

<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>

HTML ELEMENTS

Elementos HTML são os blocos de construção básicos de uma página da web.
Cada elemento possui uma tag que abre e uma tag que fecha ele.

alguns exemplos de elementos em html:

EXEMPLO

<h1> TITULO </h1>

TITULO

<p> TITULO </p>

TITULO

HTML ATTRIBUTES

Attributes são um tipo de complementação para os elementos, adicionando algumas informações extras.
Todos os elementos podem ter Attributes, e eles vem geralmente com nome do atributo e valor exemplo: href = "link"

EXEMPLO

<a href = "https://www.hltv.org/" > HLTV.ORG </a>

HLTV.ORG

<img src = "link_imagem.jpg" >

HTML PARAGRAPHS

Os paragrafos sao usando geralmente para criar grandes blocos de textos dentro do codigo.
Um dos jeitos mais facil de se organizar um texto é utilizando de <br> para o quebramento de linha e o <p> para criação de paragrafos

EXEMPLO

<p> Lorem ipsum dolor sit amet consectetur <br> Quode ducimus reiciendis impedit expedita, labore tempore eaque esse dolorem quibusdam eurum? </p>

HTML STYLES

O style é um atributo que busca trazer uma personalização e estilo para o codigo podendo atribuir cores e tamanhos diiferenciados ao codigo.

EXEMPLO

<h2 style="color:blue;"> TITULO2 </h2>

TITULO2

<h2 style="font-family:Monocraft;"> TITULO2 </h2>

TITULO2

HTML FORMATTING

A parte de formatação de texto é muito imporatante pna hora de criar um site no html podendo mudar como o texto vai aparecer na pagina

EXEMPLO

<b> TEXTO NEGRITO
<strong> TEXTO IMOPORTANTE
<i> TEXTO ITALICO
<small> texto pequeno
<mark> texto marcado
<del> texto riscado
