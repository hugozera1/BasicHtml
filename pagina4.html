HTML INPUT ATTRIBUTES

input também possui diversos tipos de atributos!:

O value serve como um especificador de valor para um input

EXEMPLO

<form> <label for="fname">First name:</label>
<input type="text" id="fname" name="fname" value="John">
<label for="lname">Last name:</label>
<input type="text" id="lname" name="lname" value="Doe">
</form>

e o input disable atributo especifca quando um input deve ficar desativado

EXEMPLO

<form> <label for="fname">First name:</label>
<input type="text" id="fname" name="fname" value="John" disabled>
<label for="lname">Last name:</label>
<input type="text" id="lname" name="lname" value="Doe">
</form>



INPUT FORM ATTRIBUTES

Este capítulo descreve os diferentes atributos form* para o elemento HTML <input>.

O Atributo form

O atributo form especifica o formulário ao qual o elemento <input> pertence.

O valor deste atributo deve ser igual ao atributo id do elemento <form> ao qual ele pertence.

EXEMPLO

<form action="/pagina_de_acao.php" id="form1">
<label for="fname">Nome:</label>
<input type="text" id="fname" name="fname"><br><br>
<input type="submit" value="Enviar">
</form> <label for="lname">Sobrenome:</label>
<input type="text" id="lname" name="lname" form="form1">

O Atributo formaction

O atributo formaction especifica a URL do arquivo que processará a entrada quando o formulário for enviado.

EXEMPLO

<form action="/pagina_de_acao.php">
<input type="text" id="fname" name="fname"><br><br>
<input type="submit" value="Enviar">
<input type="submit" formaction="/pagina_de_acao2.php" value="Enviar como Administrador">
</form>

HTML MEDIA

na hora de fazer um site você tem que levar em conta quais tipos de midia e arquivos vao estar presentes no seu site e aqui vai uma lista de alguns formatos conhecidos

EXEMPLO

  • MP3: Formato de áudio comprimido, amplamente utilizado para músicas e áudio em geral.
  • MPEG-4: Formato de vídeo comum, suporta alta qualidade e é amplamente utilizado em vídeos online, streaming e DVDs.
  • JPEG: Formato de imagem comprimida, ideal para fotografias e imagens com muitos detalhes.
  • PNG: Formato de imagem sem perdas, usado para gráficos na web e imagens com transparência.
  • GIF: Formato de imagem animada, frequentemente usado para pequenas animações e memes.
  • MP4: Formato de vídeo compactado, suporta alta qualidade e é amplamente utilizado em vídeos online e streaming.
  • FLAC: Formato de áudio sem perdas, ideal para audiófilos e arquivos de música de alta qualidade.
  • WebM: Formato de vídeo aberto, otimizado para a web e amplamente utilizado em vídeos online.
  • SVG: Formato de imagem vetorial, ideal para gráficos escaláveis e ícones.
  • AVI: Formato de vídeo antigo, ainda suportado por muitos players, mas menos eficiente que formatos modernos como MP4.
