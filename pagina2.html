HTML LINKS

Para inserir um link em uma pagina html é necessario usar a tag <a> exemplo:

EXEMPLO

<a href="https://www.hltv.org/"> hltv.org
<a href="https://www.youtube.com/"> youtube.com

HTML FAVICONS

O favicon é o icone que fica na parte superior esquerda do navegador e que serve como um icone da pagina exemplo:

EXEMPLO

<link rel="icon" href="favicon-ex.png">

HTML TABLE

No html para fazer tabelas se usa a tag <table> para a criação do corpo da table e a <th> para criação do cabeçalho da tabela

EXEMPLO

<table border="1">
<tr>
<th>Cabeçalho 1</th>
<th>Cabeçalho 2</th>
</tr>
<tr>
<td>Célula 1</td>
<td>Célula 2</td>
</tr>
</table>
Cabeçalho 1 Cabeçalho 2
Célula 1 Célula 2

HTML Block & Inline

Um elemento de bloco sempre inicia em uma nova linha, e os navegadores automaticamente adicionam algum espaço (uma margem) antes e depois do elemento.
Um elemento de bloco sempre ocupa a largura total disponível (se estende para a esquerda e para a direita o máximo que pode).
Dois elementos de bloco comumente usados são: <p> e <div>.

EXEMPLO BLOCK

<p style="border: 1px solid black">Hello World</p> <div style="border: 1px solid black">Hello World</div>

Hello World

Hello World

Um elemento inline não inicia em uma nova linha. Um elemento inline ocupa apenas a largura necessária.

EXEMPLO INLINE

<p>Esse é um inline span <span style="border: 1px solid black">Hello World</span> elemento dentro de um paragrafo</p>

This is an inline span Hello World element inside a paragraph.
